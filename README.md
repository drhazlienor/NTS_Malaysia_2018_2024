# NTS_Malaysia_2018_2024
The R scripts used for data cleaning, statistical analysis, and modelling in this study are available in a public GitHub repository. The repository is organised into folders corresponding to each stage of the analysis: 
(i) bibliometric analysis
(ii) outbreak epidemiology
(iii) incidence and AMR analysis
(iv) time-series modelling. 
Each script is annotated to facilitate reproducibility. No individual-level or identifiable patient data are shared in this repository. All data used in the analyses are either aggregated or de-identified in accordance with ethical and data governance requirements.
